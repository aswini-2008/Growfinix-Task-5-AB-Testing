# A/B Testing & Statistical Analysis

## Objective

Compare the performance of two marketing groups using statistical testing.

## Technologies Used

* Python
* Pandas
* SciPy
* Matplotlib

## Features

* Data loading and preprocessing
* A/B testing using T-Test
* Statistical significance evaluation
* Conversion rate comparison chart

## Files

* ab_testing.py
* marketing_data.csv
* screenshots/

## Output

* T-Statistic
* P-Value
* Conversion rate visualization

## Conclusion

The project demonstrates how A/B testing can be used to evaluate marketing performance and make data-driven decisions.
